Announcing EffiState 1.0

Introducing EffiState, a new generation of state management focused on speed and developer experience.

Today, we're thrilled to announce the release of EffiState 1.0, a state management library designed from the ground up for modern JavaScript applications. After months of development and testing, we're confident that EffiState sets a new standard for performance and developer experience.

A New Approach to State Management

EffiState was born out of our frustration with existing state management solutions. We wanted something that was:

  • Blazing fast - Optimized for performance in both small and large applications
  • Simple to learn - Intuitive API that developers can pick up in minutes
  • Framework agnostic - Works with React, Vue, Angular, or vanilla JavaScript
  • Lightweight - Under 2KB minified and gzipped
  • Type-safe - First-class TypeScript support

With EffiState, we've achieved all of these goals and more. Our benchmarks show that EffiState is up to 3x faster than other popular state management libraries, with a significantly smaller bundle size.

Key Features

Simplified State Updates

Unlike other state management libraries that require actions, reducers, and complex boilerplate, EffiState allows you to update state directly: 

// Create a store
const store = createStore({ count: 0 });

// Update state directly
function increment() {
  store.set({ count: store.get().count + 1 });
}

// Subscribe to changes
store.subscribe((newState, oldState) => {
  console.log('Count changed:', newState.count);
});

Optimized Array Operations

EffiState includes specialized methods for array operations that provide significant performance improvements over standard immutable updates: 

// Add an item to an array
store.array.push('todos', { id: 1, text: 'Learn EffiState', completed: false });

// Update an item
store.array.map('todos', todo => 
  todo.id === 1 ? { ...todo, completed: true } : todo
);

// Remove items
store.array.filter('todos', todo => !todo.completed);

Getting Started

Getting started with EffiState is easy. Install it from npm: 

npm install effistate

Then create your first store: 

import { createStore } from 'effistate';

// Create a store with initial state
const store = createStore({
  count: 0,
  todos: [],
  user: null
});

// Export functions to update state
export function increment() {
  store.set({ count: store.get().count + 1 });
}

// Export the store for use in your app
export default store;

Check out our Getting Started guide for more detailed instructions and examples.
