Store

The Store is the central concept in EffiState, responsible for holding your application state and providing methods to interact with it.

Creating a Store

You create a store using the createStore function, providing the initial state: 

import { createStore } from 'effistate';

const store = createStore({
  count: 0,
  todos: [],
  user: null
});

The store exposes methods to get and update the state, subscribe to changes, and create selectors.

Store Methods

get()

Retrieves the current state: 

const state = store.get();
console.log(state.count); // 0

set()

Updates the state with a partial state object: 

store.set({ count: 1 });
console.log(store.get().count); // 1

subscribe()

Subscribes to state changes: 

const unsubscribe = store.subscribe((newState, oldState) => {
  console.log('State changed:', newState);
});

// Later, to unsubscribe:
unsubscribe();

array

Optimized array operations: 

// Add an item
store.array.push('todos', { id: 1, text: 'Learn EffiState' });

// Filter items
store.array.filter('todos', todo => !todo.completed);

Store Configuration

You can configure a store with additional options: 

const store = createStore(
  { count: 0 },
  {
    immutable: true,
    enableDevTools: true,
    maxHistoryLength: 50
  }
);

See the API Reference for all available configuration options.
