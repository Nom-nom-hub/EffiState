Why EffiState?

EffiState was designed to solve common problems with state management:

  • Performance: Optimized for minimal re-renders and efficient updates
  • Simplicity: Intuitive API that's easy to learn and use
  • Flexibility: Works with any JavaScript framework or vanilla JS
  • Size: Tiny footprint (under 2KB) with zero dependencies
  • TypeScript: First-class TypeScript support with comprehensive types

Whether you're building a small project or a complex application, EffiState provides the tools you need for effective state management.

Quick Start

Install EffiState:

npm install effistate

Create your first store:

import { createStore } from 'effistate';

// Create a store with initial state
const store = createStore({
  count: 0,
  todos: [],
  user: null
});

// Get the current state
console.log(store.get().count); // 0

// Update the state
store.set({ count: 1 });

// Subscribe to changes
store.subscribe((newState, oldState) => {
  console.log('State updated:', newState);
});

For a more detailed introduction, check the Getting Started guide.
