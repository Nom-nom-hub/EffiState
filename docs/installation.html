Installation

Learn how to install EffiState in your JavaScript or TypeScript project.

Using Package Managers

npm

npm install effistate

yarn

yarn add effistate

pnpm

pnpm add effistate

Framework Integrations

EffiState provides dedicated integration packages for popular frameworks:

React

npm install effistate effistate-react

Vue

npm install effistate effistate-vue

Svelte

npm install effistate effistate-svelte

Angular

npm install effistate effistate-angular

Using CDN

You can also use EffiState directly from a CDN: 

<script src="https://cdn.jsdelivr.net/npm/effistate@latest/dist/effistate.min.js"></script>

<script>
  // EffiState is available as a global variable
  const store = EffiState.createStore({ count: 0 });
</script>

Verifying Installation

Create a simple store to verify your installation: 

import { createStore } from 'effistate';

const store = createStore({ count: 0 });
console.log(store.get()); // { count: 0 }

store.set({ count: 1 });
console.log(store.get()); // { count: 1 }

If this code runs without errors, EffiState is correctly installed.

TypeScript Support

EffiState includes built-in TypeScript typings, so no additional packages are required for TypeScript projects.
